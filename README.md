# Undercover Kids

Jeu de société adapté pour enfants (3-6 ans), inspiré du jeu Undercover. Les joueurs recoivent des emojis qu'ils doivent décrire à l'oral pour démasquer l'espion.

**Jouer** : https://a-belhadj.github.io/undercover_kids/

## Lancer en local

```bash
make up       # serveur de dev (accessible sur le réseau)
make check    # lint + typecheck + tests
make build    # build de production
```
